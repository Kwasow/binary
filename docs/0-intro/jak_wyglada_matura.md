# Orgzanizacja

Matura składa się z jednego arkusza, w którym znajdują się zarówno zadania
testowe, jak i takie do wykonania z wykorzystaniem komputera. Na całość matury
przeznaczone jest 210 minut (3.5 godziny).

# Typy zadań

Inny test

# Warto wiedzieć

Jeszcze inny test
